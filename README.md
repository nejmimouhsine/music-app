#### Built With

- [Vue](https://v3.vuejs.org/)
- [Vuex](https://vuex.vuejs.org/)
- [Firebase](https://firebase.google.com/)
- [Tailwind](https://tailwindcss.com/)

## Installation

1.Clone the repo.
2.Install packages:

```
yarn install
```

Or, if you are using npm:

```
npm install
```

3.Run:
`yarn serve`
Or:
`npm run serve`
